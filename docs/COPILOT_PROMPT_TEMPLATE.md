# Copilot Prompt Template

When asking Copilot to generate code, use this structure:

1. **Context**
   - What part of CourseForge this relates to
   - Relevant files or modules
   - Any constraints

2. **Goal**
   - What you want Copilot to produce

3. **Rules**
   - Follow CourseForge architecture
   - Follow coding style guidelines
   - Use TypeScript
   - Keep functions small and clear

4. **Output Format**
   - One file or multiple files
   - Include comments where helpful
